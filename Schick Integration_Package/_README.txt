When installing on Windows11

**EXTRACT CONTENTS OF CDR ELITE SETUP FOLDER

**The first 3 steps need to happen on every Eaglesoft Workstation install for Eaglesoft imaging.
**Order is important - must follow order to avoid crashing when opening images in Eaglesoft.

1. Install the Security KB msxml4-KB2758694-enu.exe.
2. Install the CDR Elite Driver Setup exe.
3. Install the CDRPatch-2808.msi.




-Next steps for OPs using Schick 33 Sensors.

**EXTRACT CONTENTS OF SHICK INTEGRATION FOLDER

4. Run Schick Integration EXE.
